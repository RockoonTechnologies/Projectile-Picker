ProjectilePicker API Docs

For that one person who wants to make requests

ProjectilePicker boasts a public api, so no key is needed,
so you can request as much as you want (though I would prefer if you do so withen reason)

GET the Database

Its pretty simple, if you want everything in the DB, or some entries, do:

GET- /api/main

Optional Parameters:

-amount (int) - caps the amount of entries to return

Returns:

main returns a array full of entries, a example looks like so:
{"data:
[{"name":"Apprentice","img":"https://www.apogeerockets.com/images/apogee_products/apprentice.jpg",
"manufacturer":"Apogee","diameter":25.4,"level":"LPR","motor":18.0,
"url":"https://www.apogeerockets.com/Rocket-Kits/Skill-Level-1-Model-Rocket/Kits/Apprentice","price":11.3,
"material":"","mass":28.35}, ...

Search the Database

GET- /api/search?name={name}

Search the Database for a Rocket Kit by name (eventually you can search by other parameters)

Returns:

Returns a entry, or none if it fails to find the query:
{"name":"Apprentice","img":"https://www.apogeerockets.com/images/apogee_products/apprentice.jpg",
"manufacturer":"Apogee","diameter":25.4,"level":"LPR","motor":18.0,
"url":"https://www.apogeerockets.com/Rocket-Kits/Skill-Level-1-Model-Rocket/Kits/Apprentice","price":11.3,
"material":"","mass":28.35}

Search for Motors

GET- /api/searchMotor?query={name of the motor}

Search ThrustCurve for a motor of supplied name (all this is, is a wrapper for ThrustCurve)

Returns:
